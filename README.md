# Tron Deep Learning - AI Experimentation

## 🚀 Overview
This project is a strategic experimentation designed to embrace the "AI shift" and master modern machine learning tools. It features a reproduction of the iconic **Tron Light Cycle** arcade game.

The core challenge lies in the red opponent: it is powered by **TensorFlow.js** and uses reinforcement learning to analyze every failure, refining its strategy in real-time to challenge the player.

## 🛠 Tech Stack

### Current Implementation (Web)
* **Frontend:** React / TypeScript
* **Backend:** Node.js
* **AI Engine:** TensorFlow.js

### Future Roadmap (2026)
* **Full Stack Refactor:** Transitioning to **Dart & Flutter**.
* **Objective:** Optimize performance and deliver a cross-platform mobile prototype.

## 🧠 Key Features
* **Self-Learning AI:** The red motorcycle doesn't follow a fixed script; it learns from gameplay data.
* **Real-time Adaptation:** The AI adjusts its movement patterns based on previous losses.
* **Modern Web Architecture:** Built with high-performance TypeScript to handle game logic and AI processing simultaneously.

## 🎯 Project Goal
This project serves as a bridge between traditional web development and the future of AI-integrated applications. It demonstrates the ability to manage complex state, neural network training, and real-time game loops.
